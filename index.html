Hello! I am Andrew, from Liverpool, UK and I live in Ibiza, Spain.

I love electronic music, exercising, travelling and festivals. I am intrigued by design, photography and writing, and recently became a qualified Software Developer to build websites and applications.

I graduated as a Full Stack Software Developer with a University accredited Diploma from Code Institute. I plan, design and execute web projects from start to finish, working as part of a web development team, or an enthusiastic self starter.

I build sites and applications using a range of programmes in a fresh and unique style to create a great user experience.

I have a good knowledge of code versioning and resolving code issues - with the ability to troubleshoot, debug and fix code.

My skills include having a solid grasp of HTML, CSS and JavaScript to create a brilliant User Experience Design. I am also proficient in Python, Flask, Django, MongoDB and SQL to process and present information.
